### 🦾 Robot Manipulation

| Project | Overview |
|---|---|
| **Maze Solving with MyCobot Pro 600** *(RAS-545 Robotic Systems)* | Tackled the challenge of autonomously solving **4x4 rectangular mazes** using the **MyCobot Pro 600** robotic arm and **AI Camera Kit**. A maze was generated, printed on a plastic board, and placed in the camera zone. The solution algorithm processed a camera image to identify the entrance, solution path, and exit, then translated the maze into **real-world waypoints** for the robot to navigate. Built a **Digital Twin** of the robot in MATLAB to simulate and validate the solution path using **forward and inverse kinematics** before real-world deployment. Integrated **socket programming** for seamless simulation-to-hardware execution. The final system works for any 4x4 maze generated with customized parameters. **Tech:** Python · MATLAB · URDF · AI Camera Kit · Inverse Kinematics · Path Planning · Socket Programming · MyCobot Pro 600 |
